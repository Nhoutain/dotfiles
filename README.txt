
# Installation

## Files installation

```
git clone git@bitbucket.org:nhoutain/dotfiles.git ${HOME}/dotfiles
cd ${HOME}/dotfiles
./installDotFiles
./installSoftware
```

## Vim plugins installation

Open vim and execute the command
```
:PlugInstall
```
